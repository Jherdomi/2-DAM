¿Qué es jQuery?

jQuery es una pequeña librería de JavaScript. Su utilidad recae en que facilita mucho el trabajo con documentos HTML, como recorrerlo, manipularlo, manejo de eventos, animaciones y AJAX ( Asynchronous JavaScript and XML). Funciona con la mayoría de navegadores, y la puedes descargar aquí. Los desarrolladores de jQuery dan dos opciones a la hora de descarga, una versión comprimida y otra no comprimida. La no comprimida es para usarla durante el desarrollo, y la comprimida ahorra ancho de banda y mejora el desempeño del código.

Ejemplos de jQuery

Veamos algunos ejemplos de jQuery en acción:

Recorrido de DOM y manipulación de HTML

Cambia el HTML del elemento <button> con la clase continue a "Siguiente..." 

$( "button.continue" ).html( "Siguiente..." )

Manejo de eventos

Muestra el elemento #banner-message que está oculto con display:none cuando se clica en cualquier botón con #button-container. 

var hiddenBox = $( "#banner-message" );
$( "#button-container button" ).on( "click", function( event ) {
    hiddenBox.show();
});

Ajax

Realiza una llamada a un script local en el server /api/getWeather con el parámetro zipcode=97201 y sustituye el HTML del elemento #weather-temp con el texto que devuelve el script. 

$.ajax({
    url: "/api/getWeather",
    data: {
        zipcode: 97201
    },
    success: function( result ) {
        $( "#weather-temp" ).html( "" + result + " degrees" );
    }
});

P.D. ¿Qué es AJAX?

Ajax no es una tecnología en sí, sino que se refiere a una forma de utilizar varias tecnologías existentes (HMTL/XHTML, CSS, JS, DOM, XML, XSLT) junto con lo más importante, el objeto XMLHttpRequest. Cuando estas tecnologías se combinan con AJAX, es posible crear aplicaciones web capaces de actualizarse sin necesitar recargar la página completa, mejorando la rapidez de respuesta de la página.
